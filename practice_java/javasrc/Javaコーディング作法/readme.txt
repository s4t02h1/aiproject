■ ファイルの構成
   javanosaho.xml        Checkstyleの設定ファイル
   javanosaho_fmt.xml    Java Code Style Formatterの設定ファイル


■ 設定ファイルのEclipseへの取り込み

・Checkstyleの設定ファイルの取り込み手順

【１】Eclipseの「ウィンドウ（W）」メニューから「設定（P）」を選択します。

【２】設定画面が表示されるので、画面上左側のメニューから「Checkstyle」を
　　　選択し、「新規」ボタンで、新しいチェック構成を作成します。

【３】「チェック構成プロパティー」画面が表示されるので、「型」は「内部構
　　　成」を選択し、「名前」には開発プロジェクトで取り決めたチェック構成
　　　の名前を入力して、「インポート」ボタンでjavanosaho.xmlファイルを読
　　　み込みます。

【４】さらに設定画面の「デフォルトとして設定」ボタンで、標準として採用し
　　　ます。

・Java Code Style Formatterの設定ファイルの取り込み手順

【１】Eclipseの「ウィンドウ（W）」メニューから「設定（P）」を選択します。

【２】設定画面が表示されるので、画面左側のメニューから「Java」→「コード・
　　　スタイル」→「フォーマッター」を選択します。

【３】さらに「インポート」ボタンでjavanosaho_fmt.xmlファイルを読み込み、
　　　「適用」ボタンで設定を確定します。
